使用 Gemini Pro 高效开发应用

mp.weixin.qq.comAndroid Android 开发者
作者 /工程副总裁 Dave Burke

不久前，我们推出了最强大的基础模型 Gemini。Gemini 是一个多模态模型，它既可以接受文本输入，也可以接受图像输入。我们为 Android 开发者们介绍一种方法，以帮助您在设备上使用我们最小规模的模型：Gemini Nano。Gemini Nano 可以通过 AICore 在特定设备上使用。AICore 是一种处理模型管理、运行时、安全功能等任务的系统服务，有助于简化开发者的工作。现在，我们将向您介绍访问 Gemini Pro 模型的全新方法，该模型可以在 Google 的数据中心中运行，而不是在设备上运行。

  • Gemini Nano
    https://android-developers.googleblog.com/2023/12/a-new-foundation-for-ai-on-android.html
  • AICore
    https://developer.android.google.cn/ml/aicore

使用 Gemini Pro 开发应用

Gemini Pro 可通过 Gemini API 访问，它是我们在各种文本和图像推理任务中进行扩展的最佳模型。为了简化 Gemini Pro 的集成，您可以使用适用于 Android 的客户端 SDK:Google AI SDK。该 SDK 支持从 Android 应用直接集成，无需开发者构建和管理自己的后端基础设施，从而降低开发成本并提高开发速度。

  • Gemini API
    https://ai.google.dev/?utm_source=android&utm_medium=referral&utm_campaign=blog&utm_content

  • Google AI SDK
    http://ai.google.dev/tutorials/android_quickstart/?utm_source=android&utm_medium=referral&utm_campaign=blog&utm_content

Google AI Studio 可提供一种简化的方式，帮助开发者集成 Gemini Pro 模型、创建提示、生成 API 密钥，并轻松地将创意落地为 AI 应用。在 Google AI Studio 中开发好提示后，您只需点击 "获取代码" 操作，即可生成一个 Kotlin 代码片段，然后便可立即使用适用于 Android 的 Google AI SDK 开始集成 Gemini。

  • Google AI Studio
    https://ai.google.dev/available_regions?hl=zh-cn

用于 Gemini API 的 Kotlin 代码

我们还在最新的 Android Studio 预览版中为开发者直接使用 Gemini API 提供了更简便的方式。即将推出的新项目模板可帮助开发者立即开始使用适用于 Android 的 Google AI SDK。借助 Android Studio 的增强型代码补全和 Lint 检查工具，您可以更轻松地处理与 API 密钥和安全性相关的问题。

  • 最新的 Android Studio 预览版
    https://developer.android.com/studio/preview

  • 新项目模板
    https://developer.android.google.cn/studio/preview/gemini-template

△ Android Studio 中 AI 的新项目模板

如果您想使用 Android Studio 中的新模板，可以通过 "文件">"新建">"新项目"来选择 Gemini API 入门模板。此模板为预配置项目提供了使用 Gemini API 所需的代码。在您选择项目名称和位置后，系统将提示您在 Google AI Studio 中生成 API 密钥，并要求您在 Android Studio 中输入该密钥。Android Studio 将通过与 Gemini API 的连接自动为您设置项目，从而简化您的工作流程。

  • 生成 API 密钥
    https://makersuite.google.com/app/apikey/?utm_source=android&utm_medium=referral&utm_campaign=blog&utm_content

或者，您可以通过导入生成式 AI 代码示例，并在 Android Studio 中通过选择 "文件">"新建">"导入示例"，然后搜索 "生成式 AI 示例"进行设置。

  • 生成式 AI 代码示例
    https://github.com/google/generative-ai-android/tree/main/generativeai-android-sample
